## Getting Started

* Clone this repo <https://github.com/ksigei/oop_school_library.git>

    ```bash
    git clone https://github.com/ksigei/oop_school_library.git
    ```

* Navigate to oop_school_library folder/directory

    ```bash
    cd folder-name
    ```

* Install gems

    ```bash
    bundle install
    ```
